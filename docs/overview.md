# BankAI Overview

BankAI is a banking guidance assistant, not a core banking platform.

## Scope

- banking process guidance
- banking forms and field explanation
- ATM usage guidance
- KYC and document awareness
- summaries of indexed or uploaded banking documents

## Out of Scope

- live account data
- transactions
- personal account lookup
- real-time bank actions

## Retrieval Rule

If the indexed or uploaded material does not contain the answer, BankAI should explicitly say so.
