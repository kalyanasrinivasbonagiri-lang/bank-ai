# Data Guidelines

Place banking source documents in `data/raw/` using topic folders:

- `deposits/`
- `withdrawals/`
- `atm/`
- `balance/`
- `forms/`
- `kyc/`
- `rules/`
- `accounts/`

Supported source formats for indexing:

- `.txt`
- `.pdf`

Supported upload formats:

- `.txt`
- `.pdf`
- `.png`
- `.jpg`
- `.jpeg`

Suggested content examples:

- deposit instructions
- withdrawal slips
- ATM usage leaflets
- bank account balance enquiry instructions
- KYC document requirement notes
- RBI or bank guideline extracts, only when permitted and locally stored
